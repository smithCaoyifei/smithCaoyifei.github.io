属性的简洁表示法

es6允许你直接写入变量和函数，作为对象的属性和方法

即es6允许在对象之中只写属性名，不写属性值。此时属性值等于属性名所代表的变量

同时方法也可以简写

属性名表达式

es6允许你字面量定义对象时，用表达式作为对象的属性名，即把表达式放在方括号内

表达式还可以用于定义方法名

注意：属性名表达式与简洁表示法，不能同时使用

Object.is()

es5比较两个值是否相等，只有两个运算符，== 和 ===，前者会自动转换数据类型，后者的NaN不等于自身，以及+0等于-0

Object.is()比较两个值是否严格相等，与===基本一致，不同之处只有两处

  • +0不等于-0
  • NaN等于自身

Object.assign()

Object.assign()方法用于对象的合并，将源对象的所有可枚举属性，复制到目标对象(target)

Object.assign方法的第一个参数是目标对象，后面的参数都是源对象。

如果目标对象与源对象有同名属性，或多个源对象有同名属性，则后面的属性会覆盖前面的属性

如果只有一个参数，Object.assign会直接返回该参数。

如果该参数不是对象，Object.assign()，则会先转成对象，然后返回

由于undefined和null无法转化对象，所以如果它们作为参数，就会报错

如果非对象参数出现在源对象的位置(即非首参数)，处理规则不同，这些参数都会被转化为对象，如果无法转化为对象，则会被跳过。即意味undefined和null不在首参数，就不会报错

object.assign拷贝的属性是有限制的，只拷贝源对象的自身属性，不拷贝继承属性，也不拷贝不可枚举的属性

Object.assign方法实行的是浅拷贝，而不是深拷贝，也就是说，如果源对象某个属性的值是对象，那么目标对象拷贝得到的是这个对象的引用

对于嵌套的对象，一旦遇到同名属性，Object.assign的处理方法是替换，而不是添加。

常见用途

  1. 为对象添加属性
  2. 为对象添加方法
  3. 克隆对象：不过采用这种方法克隆，只能克隆原始对象自身的值，不能克隆它继承的值
  4. 合并多个对象
  5. 为属性指定默认值
  6. 属性的可枚举性

属性的可枚举性

对象的每个属性都有一个描述对象，用来控制该属性的行为。Object.getOwnPropertyDescriptor(obj, 'foo') 可以获取该属性的描述对象。

描述对象的enumerable属性，称为”可枚举性”,如果该属性为false,就表示某些操作会忽略当前属性,即该操作遍历时不会返回该属性

es5中有三个操作会忽略enumerable为false的属性

  1. for…in循环：只便利对象自身和继承的可枚举属性
  2. Object.keys()：返回对象自身的所有可枚举的属性的键名
  3. JSON.stringify()：只串行化对象自身的可枚举的属性

es6中新增了两个操作，会忽略enumerable为false的属性

  1. Object.assign()：只拷贝对象自身的可枚举的属性
  2. Reflect.enumerate()：返回所有for…in循环会遍历的属性

在这五个操作中，只有for…in和Reflect.enumerate()会返回继承的属性。

引入enumerable的最初目的，就是让某些属性可以规避掉for…in操作。比如，对象原型的toString方法，以及数组的length属性，就是通过这种手段，不会被for…in遍历到

所以，es6规定所有class的原型方法都是不可枚举的

引入继承的属性会让问题复杂化，所有尽量使用Object.keys()来代替for…in循环

属性的遍历

  1. for…in
  2. object.keys(obj)
  3. Object

proto属性

  • proto属性
    用来读取或者设置当前对象的prototype对象
    1. Object.setPrototypeOf()：写操作
    2. Object.getPrototypeOf()：读操作
    3. Object.create()：生成操作
  • Object.setPrototypeOf()
  • Object.getPrototypeOf()
    用于读取一个对象的prototype的对象

Object.values()和Object.keys()

  • Object.keys(obj)返回一个数组，成员是参数对象自身的(不含继承的)所有可遍历(enumerable)属性的键名
  • Object.values(obj) 返回一个数组，成员是参数对象自身(不含继承)所有可遍历的属性的键值(未完全支持！！)
  • Object.entries(obj) 返回一个数组，成员是参数对象自身(不含继承)的所有可遍历属性的键值对数组

对象的拓展运算符

es7提案，将Rest解构赋值/扩展运算符(…)引入对象

  • Rest解构赋值
    用于从一个对象取值，相当于将所有可遍历，但尚未读取的属性，分配到指定属性上。所有的键和它们的值，都会拷贝到新对象上面。

由于Rest解构赋值要求等号右边是一个对象，如果等号右边是一个undefined或者null，就会报错，因为它们无法转化为对象

Rest解构赋值必须是最后一个参数，否则会报错

Rest解构赋值的拷贝是浅拷贝，即如果一个键的值是复合类型的值(数组，函数，对象)，那么Rest解构赋值拷贝的是这个值得引用，而不是这个值的副本

Rest解构赋值不会拷贝继承自原型对象的属性

  • 扩展运算符 es7提案，es6中未支持
    扩展运算符(…)用于取出参数对象的所有可遍历属性，拷贝至当前对象中

    如果用户自定义的属性，放在拓展运算符后面，则拓展运算符内部的同名属性会被覆盖
    如果拓展运算符的参数是null,或者undefined，则这两个值会被忽略，不会报错

Object.getOwnPropertyDescriptor

es5中，Object.getOwnPropertyDescriptor方法，就是返回某个对象属性的描述对象
